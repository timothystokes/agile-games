## Architecture Constraints

- Shared libraries are stateless pure functions; cache and state are owned by calling layers.
- Library functions take explicit required parameters directly in their signatures — no callbacks or hidden context.
- Tiny single-use helpers are scoped inside the function that uses them, not top-level utilities.
